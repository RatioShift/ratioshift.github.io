New Tool: Fluid Scale Calculator Released

Published on Jan 29, 2026 ‚Ä¢ schedule

We are excited to announce the launch of our first official tool at RatioShift: The Fluid Scale Calculator.

Responsive design has evolved. We're moving away from fixed breakpoints and toward "Fluid Design," where typography and spacing scale smoothly between screen sizes. However, calculating the clamp() math manually is a headache. That‚Äôs why we built this tool.

Why Fluid Scaling?

Traditional media queries create "jumps" in your design. With fluid scaling, your font size might be 16px on a phone and 48px on a desktop, but it transitions perfectly for every pixel in between.

Key Feature: Our calculator generates the exact CSS clamp() function you need, including the viewport-based slope calculation.

How to use the tool

  • Define your range: Set your minimum and maximum font sizes.
  • Set your viewports: Tell the tool at what screen width the scaling should start and stop.
  • Copy & Paste: Grab the generated code and drop it into your CSS variables.
